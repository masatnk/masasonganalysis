    Detailed Description

    The MainWindow class inherits QMainWindow to provide ui and basic computation on wav and data.

    This class generates mainwindow and related ui components. This class can open WAV files. This class can analyze data and save them to mydata.

    Note: Only one WAV file can be hundled at a time. Analyzed data are vectors of double.

    See also main, QChartViewWAV, and QWidgetWAVChart.
